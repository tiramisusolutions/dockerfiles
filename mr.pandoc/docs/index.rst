=========
mr.pandoc
=========

.. toctree::
   :maxdepth: 2

`Pandoc <http://pandoc.org/>`_ in a `Docker <https://www.docker.com>`_ container based on `Debian "jessie" <https://www.debian.org/releases/jessie/>`_.

Pre-configured for easy converting a bunch of *rst* [reStructuredText] files into md [Markdown].

mr.pandoc is written as part of a *write and test documentation framework* for documentation based on `Sphinx <http://www.sphinx-doc.org/en/stable/index.html>`_ and used within the `Plone <https://plone.org>`_ Community.
